# Expense & Budget Frontend — Unified Stitch Layout

This build keeps the original Stitch page designs and adds:

- `my-face.html` from the latest uploaded Stitch export
- One shared header for every page
- One shared desktop and mobile navbar for every page
- Working `My Face` links in both navigation areas
- Active-page highlighting
- Profile icon in the header

## Open with Live Server

1. Extract the ZIP.
2. Open the folder in VS Code.
3. Open `index.html`.
4. Right-click and choose **Open with Live Server**.

The shared app shell is located in:

- `assets/css/layout.css`
- `assets/js/layout.js`
